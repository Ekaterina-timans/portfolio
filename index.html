Описание:

Приложение для поиска и управления рецептами, которое предоставляет пользователям возможность находить рецепты, просматривать их детали, сохранять в избранное и использовать различные фильтры для поиска подходящих рецептов.

  • Функция поиска: Позволяет пользователям искать рецепты по ключевым словам, ингредиентам или другим критериям.
  • Фильтрация: Возможность применения фильтров по категориям, типам кухни, калориям, времени приготовления, типам блюд и непереносимости продуктов.
  • Детали рецепта: Информация о рецепте, включая ингредиенты, калории, жиры, белки и углеводы, а также инструкция, которая включает в себя подробные шаги по приготовлению рецепта.
  • Сохранение рецептов в избранное и его просмотр: Возможность добавлять рецепты в список избранных для быстрого доступа.
  • Типы блюд, типы кухни и непереносимость продуктов: Фильтрация рецептов по типам блюд (например, завтрак, обед, ужин), типам кухни (например, итальянская, мексиканская) и учет ограничений в рационе, включая аллергенные продукты.
  • Пользовательский интерфейс: Дизайн приложения, созданный на основе макета в Figma с некоторыми корректировками и добавлением старницы, обеспечивающий удобный и интуитивно понятный доступ к функциям приложения.
  • Адаптивность: Оптимизация пользовательского интерфейса для различных устройств и экранов.
  • Spoonacular API: Использование Spoonacular API для получения данных о рецептах, включая ингредиенты, инструкции и информацию о питательных веществах.

Технологии и инструменты:

  • jQuery
  • CSS
  • Bootstrap
  • Webpack

Описание:

Веб-планировщик задач, который позволяет пользователю системы вести списки своих задач.

  • Организация задач: Создание, редактирование и удаление задач, а также установление их приоритетов и даты окончания.
  • Планирование: Возможность просмотра задач на различные временные промежутки — день, неделю, месяц, год.
  • Управление файлами: Возможность прикрепления файлов к задачам.
  • Поиск: Поиск задач по ключевым словам.
  • Управление пользователями: Регистрация и авторизация пользователей, а также возможность администрирования их аккаунтов.
